# Sistema Interno de Reboque — MVP V6

## Correções desta versão
- Placa enviada separadamente assim que o motorista chega na origem.
- A placa muda o serviço para "na origem" e aparece na central.
- Fotos/checklist enviadas separadamente sem finalizar o serviço.
- Finalização separada, para ser usada só no destino.
- A tela do motorista não apaga o campo enquanto ele está digitando ou escolhendo fotos.

## Como rodar
1. Extraia a pasta.
2. Clique duas vezes em `rodar_sistema.bat`.
3. No computador da central, abra: http://localhost:8000
4. No celular do motorista, use o link que aparece na tabela de motoristas.
