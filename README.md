# Sistema Interno de Reboque — MVP V2

## O que tem nesta versão
- Motorista online/offline pelo celular
- Atualização de localização via GPS do celular
- Mapa da central com motoristas online
- Cadastro de motoristas e serviços
- Envio de serviço para o celular do motorista
- Tela do motorista com aceitar/recusar
- Status: aceito, a caminho, na origem, em transporte, finalizado
- Link de rota para Google Maps

## Como rodar
1. Extraia a pasta.
2. Clique duas vezes em `rodar_sistema.bat`.
3. No computador da central, abra: http://localhost:8000
4. No celular do motorista, use o link que aparece na tabela de motoristas.

## Importante
Para o celular acessar, ele precisa estar na mesma rede Wi-Fi do computador. Use o endereço de rede mostrado no terminal, por exemplo:
http://192.168.0.42:8000

O motorista precisa permitir localização no navegador.
